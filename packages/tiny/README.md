
## 介绍
图片压缩
## 使用
1. 安装
```
npm i @llhe/tiny -g
```
2. 查看版本
```
mini --version 
// or
mini -v
```
3. 压缩目录下所有图片【支持jpg|jpeg|png格式】
```
mini
```
4. 压缩目录下指定图片
```
mini --imgs a.png
// 简写
mini -i a.png
```
